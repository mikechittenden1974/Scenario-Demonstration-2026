# Scenario Demonstration 2026

## Overview

This repository contains training resources designed to help **you practise real-world data tasks through structured scenarios**.

Instead of only reading about techniques, these materials allow you to **work through practical situations** similar to those you might encounter in a real workplace. Each scenario presents a problem, provides the relevant data, and asks you to complete a set of tasks using appropriate tools.

The aim is to help you build confidence by applying your skills to realistic examples.

---

## How These Scenarios Work

Each scenario will normally include:

- A **short background description** explaining the situation  
- A **task description** outlining what you need to do  
- A **data pack** containing the files you will work with  
- Clear instructions on what outputs you should produce  

You will typically be expected to:

- Import data from files such as **CSV, Excel, or PDF**
- Review the structure of the data
- Identify any quality or formatting issues
- Carry out analysis or preparation tasks
- Explain or present your findings

The focus is on **demonstrating how you approach a task**, not just producing the final answer.

---

## Repository Structure

As more scenarios are added, the repository will contain:

- **Scenario instructions** – the document explaining the task  
- **Data packs** – the files required for the activity  
- **Supporting notes** – additional guidance where needed  

Each scenario can be completed independently.

---

## Skills You Will Practise

Depending on the scenario, you may practise skills such as:

- Importing data from multiple file types
- Using **Excel and Power Query**
- Checking and cleaning data
- Organising datasets
- Carrying out analysis
- Interpreting results
- Presenting findings clearly

---

## Important Note

All data provided in these exercises is **simulated training data**.  
It is designed for learning purposes only and does not contain real personal or confidential information.
